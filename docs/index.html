Web page to accompany the paper: Some right ways to analyze (psycho)linguistic data, by Shravan Vasishth

The paper and reproducible code

Please note that the Rmd/html files will differ slightly from the Rnw file that produced the final pdf of the paper.
  1. The paper (pdf)
  2. The source code (knitr, Rnw format) of the paper.
  3. Rmd code (please install bcogsci first)
  4. To extract the R code from the Rmd file above, type knitr::purl("ARLVasishthSupplement.Rmd") in the current working directory within the R console.
  5. The compiled html code from the above Rmd file.
